@witchcraft/ui is an opinionated, batteries-included [Vue](https://vuejs.org)/[Nuxt](https://nuxt.com/) component library built on [Reka UI](https://reka-ui.com/) with [Tailwind CSS](https://tailwindcss.com/). Accessibility-first, themable via [metamorphosis](https://github.com/alanscodelog/metamorphosis).

While the docs have storybook embeds, all full interactive examples and demos are available in [Storybook](https://witchcraftjs.github.io/ui/storybook).
